Introducción al Desarrollo Web

El desarrollo web combina creatividad, lógica y herramientas tecnológicas para construir experiencias digitales interactivas. Conocer cómo funcionan HTML, CSS y JavaScript permite a los analistas de sistemas diseñar y validar prototipos funcionales para proyectos reales.

Desde páginas informativas hasta sistemas de gestión empresarial, estas tecnologías permiten estructurar la información, definir estilos visuales atractivos e implementar lógica que responde a las acciones del usuario.

HTML - Lenguaje de Marcado

HTML organiza el contenido en una estructura lógica, facilitando la comprensión tanto para navegadores como para desarrolladores. Utiliza etiquetas semánticas que describen el tipo de contenido que envuelven.

Ejemplo básico:

<header>
  <h1>Mi Sitio Web</h1>
</header>
<main>
  <p>Contenido principal.</p>
</main>
<footer>
  <p>Pie de página.</p>
</footer>

Elementos clave:

  • <a> para enlaces
  • <img> para imágenes
  • <form> para formularios
  • <table> para tablas de datos
  • Atributos como class, id, href, src

CSS - Diseño Visual

CSS permite definir estilos de forma centralizada. Esto mejora la mantenibilidad y promueve la separación entre la estructura del contenido y su presentación visual.

Ejemplo práctico:

h1 {
  color: #2980b9;
  text-align: center;
}

.container {
  display: flex;
  justify-content: space-around;
}

Técnicas modernas:

  • Flexbox: para alinear elementos de manera flexible.
  • Grid: para crear diseños en cuadrícula.
  • Media queries: para adaptar el diseño a diferentes dispositivos.
  • Animaciones: con @keyframes y transition.

JavaScript - Interactividad

JavaScript es esencial para cualquier sitio web dinámico. Permite manipular el contenido, reaccionar a eventos y comunicarse con servidores sin recargar la página.

Ejemplo de interacción:

document.querySelector('button').addEventListener('click', () => {
  alert('¡Hola desde JavaScript!');
});

Aplicaciones comunes:

  • Validación de formularios
  • Creación de galerías de imágenes
  • Notificaciones al usuario
  • Petición de datos usando fetch()

Integración de Tecnologías

HTML, CSS y JavaScript funcionan de forma conjunta para crear experiencias completas:

  • HTML define la estructura y el contenido.
  • CSS da estilo y presenta visualmente ese contenido.
  • JavaScript le otorga vida, reaccionando al usuario y manipulando la estructura.

Ejemplo de integración:

<h2 id="mensaje">Hola usuario</h2>
<button onclick="actualizarTexto('mensaje', 'Texto actualizado con JavaScript')">Actualizar</button>
